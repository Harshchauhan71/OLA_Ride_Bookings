# OLA_Ride_Bookings

🚖 OLA Dashboard using SQL
This project showcases an interactive OLA ride analytics dashboard built using SQL for data extraction and transformation. The dashboard provides insights into booking trends, revenue performance, ride volume, and cancellation patterns.

📊 Features
Total Bookings & Revenue Overview

Booking Status Breakdown (Success, Driver Canceled, Customer Canceled, Driver Not Found)

Ride Volume Over Time Analysis

Date Range Filtering for Dynamic Insights

User-friendly Dashboard Interface

🛠️ Tech Stack
SQL (Data querying & transformation)

Data Visualization Tool (Power BI)

📌 Key Insights
Track daily ride demand patterns

Identify cancellation trends and reasons

Analyze overall business performance
